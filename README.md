# 🚲 Bike Sharing Data Analysis Dashboard

Selamat datang di proyek akhir analisis data saya! Proyek ini mengeksplorasi faktor-faktor yang mempengaruhi penyewaan sepeda menggunakan **Bike Sharing Dataset**.

---

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis bagaimana variabel eksternal seperti **cuaca** dan **tren pertumbuhan bulanan (2011-2012)** mempengaruhi jumlah penyewaan sepeda. Hasil analisis disajikan dalam bentuk dashboard interaktif menggunakan **Streamlit**.

## 📂 Struktur Direktori
- **dashboard/**: Berisi file utama dashboard (`dashboard.py`) dan data hasil olahan (`main_data.csv`).
- **data/**: Berisi dataset mentah asli dari sumber data.
- **notebook.ipynb**: Dokumentasi lengkap proses analisis (Gathering, Assessing, Cleaning, EDA, Visualisasi).
